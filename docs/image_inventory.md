- `chr_how_cheering.png`와 `chr_how_cheering.png.png`는 이름상 중복 이미지일 가능성이 있습니다.
- `bg_intro.jpg`와 `chr_main_happy.png`는 파일 용량이 26바이트로 매우 작아 손상 파일 또는 자리표시자일 가능성이 있습니다.
